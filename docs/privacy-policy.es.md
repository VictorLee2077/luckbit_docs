# Política de privacidad de Luckbit Wallet

Última actualización: 2026-01-31

Esta Política de privacidad se aplica a la extensión de navegador Luckbit Wallet ("la Extensión"). La Extensión es una cartera no custodial. Las claves privadas se cifran y se almacenan localmente. No custodiamos ni accedemos a sus claves privadas o activos.

## 1. Información que recopilamos
**No recopilamos, almacenamos ni vendemos** información de identificación personal.  
No se requiere una cuenta ni inicio de sesión para usar las funciones principales.

## 2. Almacenamiento local de datos
Para proporcionar la funcionalidad de la cartera, la Extensión guarda **localmente en su dispositivo** los siguientes datos (no se suben):
- Claves privadas / frases semilla cifradas
- Configuración de cuentas y redes
- Información de tokens y caché del historial de transacciones
- Configuraciones del usuario (idioma, bloqueo automático, etc.)

Todos los datos sensibles se cifran localmente. La Extensión no puede leer claves privadas o frases semilla en texto plano.

## 3. Comunicación con servicios de terceros
Para consultar la cadena o transmitir transacciones, la Extensión se comunica con nodos blockchain y exploradores de bloques. Estas solicitudes pueden incluir:
- Direcciones de billetera
- Hash de transacciones
- Red e ID de cadena
- Dirección IP del dispositivo (como parte normal de la solicitud de red)

Esta comunicación solo se utiliza para la funcionalidad de la cartera. **No** compartimos ni vendemos sus datos a terceros.

## 4. Permisos (relacionados con la privacidad)
- `storage` / `unlimitedStorage`: almacenar datos y configuraciones del monedero localmente
- `tabs` / `activeTab`: interactuar con la página DApp actual, autorizar conexiones, abrir páginas necesarias
- `alarms`: temporizador de bloqueo automático
- `<all_urls>` permiso de host: inyectar el Provider y gestionar autorización en cualquier sitio DApp

## 5. Seguridad de los datos
Usamos cifrado local y bloqueo automático para proteger los datos. Guarde su frase semilla y contraseña de forma segura; no se pueden recuperar si se pierden.

## 6. Eliminación de datos
Puede restablecer la cartera o borrar los datos de la extensión para eliminar todos los datos locales. Esta acción es irreversible; haga una copia de seguridad de su frase semilla antes.
