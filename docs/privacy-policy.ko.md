본 개인정보 처리방침은 Luckbit Wallet 브라우저 확장(이하 "본 확장")에 적용됩니다. 본 확장은 비수탁형 지갑이며, 개인 키는 로컬에서 암호화되어 저장됩니다. 우리는 개인 키나 자산을 보관하거나 접근하지 않습니다.

## 1. 수집하는 정보
당사는 **개인 식별 정보를 수집, 저장, 판매하지 않습니다**.  
본 확장은 계정 등록이나 로그인 없이 핵심 기능을 사용할 수 있습니다.

## 2. 로컬 데이터 저장
지갑 기능 제공을 위해 본 확장은 **사용자 기기 로컬**에 다음 데이터를 저장합니다(업로드하지 않음):
- 암호화된 개인 키/시드 문구
- 계정 및 네트워크 설정
- 토큰 정보 및 거래 기록 캐시
- 사용자 설정(언어, 자동 잠금 등)

모든 민감한 데이터는 로컬에서 암호화되며, 확장은 평문 개인 키 또는 시드 문구를 읽을 수 없습니다.

## 3. 제3자 서비스와의 통신
블록체인 조회 및 거래 전송을 위해 본 확장은 노드/블록 탐색기 등 제3자 서비스와 통신합니다. 이 요청에는 다음이 포함될 수 있습니다:
- 지갑 주소
- 거래 해시
- 네트워크 및 체인 ID
- 기기 IP 주소(일반적인 네트워크 요청에 의해 발생)

이 통신은 지갑 기능 제공을 위한 것이며, 당사는 제3자에게 데이터를 공유하거나 판매하지 않습니다.

## 4. 권한 안내(개인정보 관련)
- `storage` / `unlimitedStorage`: 지갑 데이터 및 설정의 로컬 저장
- `tabs` / `activeTab`: 현재 DApp 페이지와 상호작용, 연결 승인, 필요한 페이지 열기
- `alarms`: 자동 잠금 타이머
- `<all_urls>` 호스트 권한: 모든 DApp에서 Provider를 주입하고 권한 처리를 수행

## 5. 데이터 보안
로컬 암호화와 자동 잠금으로 데이터를 보호합니다. 시드 문구와 비밀번호는 안전하게 보관해 주세요. 분실 시 복구할 수 없습니다.

## 6. 데이터 삭제
확장 내 "지갑 초기화" 또는 확장 데이터 삭제를 통해 로컬 데이터를 제거할 수 있습니다. 삭제 후 복구는 불가능하므로, 사전에 시드 문구를 백업해 주세요.
