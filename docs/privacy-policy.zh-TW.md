本隱私政策適用於 Luckbit Wallet 瀏覽器擴充功能（以下簡稱「本擴充」）。本擴充為非託管錢包，私鑰僅在本機加密保存，我們不託管、也不接觸您的私鑰或資產。

## 1. 我們蒐集哪些資訊
我們**不會蒐集、儲存或販售**任何可識別個人身份的資訊。  
本擴充不需要註冊帳號，也無需登入即可使用核心功能。

## 2. 本地資料儲存
為提供錢包功能，本擴充會在**使用者本機**保存以下資料（僅本地儲存，不上傳）：
- 加密後的私鑰／助記詞
- 帳戶與網路設定
- 代幣資訊與交易紀錄快取
- 使用者設定（語言、自動鎖定等）

所有敏感資料均在本機加密保存，擴充無法讀取明文私鑰或助記詞。

## 3. 與第三方服務的通訊
為查詢鏈上資料與廣播交易，本擴充需與區塊鏈節點／區塊瀏覽器等第三方服務通訊。這些請求可能包含：
- 錢包地址
- 交易雜湊
- 網路與鏈 ID
- 使用者裝置的 IP 位址（正常網路請求產生）

這些通訊僅用於錢包功能，本擴充**不會**與第三方分享或販售您的資料。

## 4. 權限說明（隱私相關）
- `storage` / `unlimitedStorage`：本地保存錢包資料與設定
- `tabs` / `activeTab`：與當前 DApp 頁面互動、連線授權與開啟必要頁面
- `alarms`：自動鎖定計時器
- `<all_urls>` 主機權限：在任意 DApp 網站注入 Provider 並完成授權流程

## 5. 資料安全
我們採用本機加密與自動鎖定機制保護資料。請妥善保管助記詞與密碼，遺失後無法找回。

## 6. 資料刪除
您可在擴充內「重置錢包」或清除擴充資料，以刪除本機所有錢包資訊。刪除後無法復原，請先備份助記詞。
